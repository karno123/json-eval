# JSON EVAL 
JSON EVAL is simple json evaluator that may be simplify business rule. 

##Operator 
Operator supported 
| No.      | Operator  |
| ---      | ---       |
| 1        | <         |
| 2        | <=        |
| 3        | >         |
| 4        | >=        |
| 5        | !=        |
| 6        | \||        |
| 7        | &&        |
